# calc

Консольный калькулятор на Go. Принимает два числа и оператор как аргументы командной строки.

## Использование

```bash
./calc <число> <оператор> <число>
```

## Примеры

```bash
./calc 10 + 5     # 15
./calc 20 - 3     # 17
./calc 6 * 7      # 42
./calc 10 / 3     # 3.333333333333333
./calc 2.5 + 1.5  # 4
```

## Поддерживаемые операторы

| Оператор | Операция       |
|----------|----------------|
| `+`      | Сложение       |
| `-`      | Вычитание      |
| `*`      | Умножение      |
| `/`      | Деление        |

## Ошибки

```bash
./calc 10 / 0
# ошибка: деление на ноль

./calc 10 ^ 2
# ошибка: неизвестный оператор "^"

./calc 10 5
# ошибка: нужно ровно 3 аргумента (получено: 2)

./calc abc + 5
# ошибка: "abc" не является числом
```

## Установка и запуск

```bash
git clone https://github.com/Shipovmax/calc
cd calc
go build -o calc .
./calc 10 + 5
```

## Стек

- Go 1.26+
