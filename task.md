# Проект 1 — Калькулятор CLI

## Что должно работать

```bash
./calc 10 + 5    # 15
./calc 20 - 3    # 17
./calc 6 * 7     # 42
./calc 10 / 3    # 3.333...
./calc 10 / 0    # ошибка: деление на ноль
./calc 10 ^ 2    # ошибка: неизвестный оператор
./calc 10        # ошибка: неверное кол-во аргументов
```

## Требования

**Обязательно:**
- Принимает ровно 3 аргумента: число, оператор, число
- Операторы: `+` `-` `*` `/`
- Числа — float64 (чтобы `10 / 3` давало `3.3333`, а не `3`)
- Любая ошибка — понятное сообщение в stderr + exit code 1
- Весь код в одном файле `main.go`

**Запрещено:**
- Сторонние библиотеки — только стандартная либа
- `fmt.Println` для ошибок — используй `fmt.Fprintf(os.Stderr, ...)`

**Темы которые ты должен использовать:**
- `os.Args` — получить аргументы
- `strconv.ParseFloat` — конвертировать строку в число
- `switch` — выбор операции
- `fmt.Fprintf(os.Stderr)` — вывод ошибок
- `os.Exit(1)` — завершение с ошибкой
